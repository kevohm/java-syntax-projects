# Java Syntax Practice Projects

## Overview

This repository contains a collection of small, focused Java projects designed to build a strong foundation in core Java syntax and programming concepts. Each project targets specific areas such as control flow, object-oriented programming (OOP), collections, and exception handling.

These projects are intentionally simple but structured to reinforce clean coding practices and problem-solving skills.

---

## Projects Included

### 1. CLI Calculator

**Description:**
A command-line calculator that performs basic arithmetic operations.

**Features:**

* Supports +, -, *, /, %
* Handles invalid input
* Modular method-based design

**Concepts Covered:**

* Control flow (if/switch)
* Methods
* Input parsing

---

### 2. Number Analyzer

**Description:**
Analyzes a given number and provides multiple outputs.

**Features:**

* Checks if number is prime
* Determines even/odd
* Calculates factorial
* Generates Fibonacci sequence

**Concepts Covered:**

* Loops (for, while)
* Conditional logic
* Method decomposition

---

### 3. Student Management System (CLI)

**Description:**
A simple system to manage student records via command line.

**Features:**

* Add student
* Search by ID
* Delete student

**Concepts Covered:**

* Classes and objects
* ArrayList usage
* Basic CRUD operations

---

### 4. Bank Account System

**Description:**
Simulates basic banking operations using OOP principles.

**Features:**

* Deposit and withdraw
* Account types (e.g., SavingsAccount)
* Interest calculation

**Concepts Covered:**

* Inheritance
* Encapsulation
* Method overriding

---

### 5. To-Do List (CLI)

**Description:**
A task manager for tracking daily activities.

**Features:**

* Add and remove tasks
* Mark tasks as complete
* List all tasks

**Concepts Covered:**

* Collections (List, Map)
* Generics
* Iteration

---

### 6. Word Frequency Counter

**Description:**
Counts how many times each word appears in a text.

**Features:**

* Input text processing
* Outputs word frequency

**Concepts Covered:**

* HashMap
* String manipulation
* Looping constructs

---

### 7. File Reader Utility

**Description:**
Reads content from files and handles errors gracefully.

**Features:**

* File reading
* Error handling for missing files

**Concepts Covered:**

* Exception handling (try-catch-finally)
* Java IO

## How to Run

1. Clone the repository:

```
git clone https://github.com/kevohm/java-syntax-projects.git
```

2. Navigate to a project folder:

```
cd calculator
```

3. Compile the Java file:

```
javac Main.java
```

4. Run the program:

```
java Main
```
